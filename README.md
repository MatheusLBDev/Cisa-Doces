# 🍰 Projeto: Página de Vendas de Doces

## 📌 Visão Geral

Este projeto tem como objetivo desenvolver uma **página de vendas online de doces**, focada em apresentar produtos de forma atrativa, facilitar pedidos.

A solução busca melhorar a experiência do cliente, aumentar a conversão de vendas e otimizar o processo de pedidos.

---

## 🎯 Objetivos

- Exibir catálogo de doces com fotos e descrições
- Informar preços de forma clara
- Permitir pedidos rápidos
- Automatizar atendimento
- Implementar cálculo automático
- Possibilitar análise e previsão de vendas

---

## 👥 Público-Alvo

- Clientes locais
- Consumidores que desejam encomendar doces para festas
- Pessoas que buscam praticidade para comprar
